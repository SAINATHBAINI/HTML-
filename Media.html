1.video: The video tag is used to embed video files in an HTML document. It supports multiple attributes to control the video playback. Ex: *Attributes for video Tag -src: Specifies the path to the video file. -controls: Adds video controls, like play, pause, and volume. -autoplay: Automatically starts playing the video when the page loads. -loop: Repeats the video once it ends. -muted: Mutes the video by default. -poster: Specifies an image to be displayed before the video starts playing. -width and height: Specifies the dimensions of the video. 2.Audio: -The audio tag is used to embed audio files in an HTML document. It also supports multiple attributes for control. Ex: *Attributes for audio Tag -src: Specifies the path to the audio file. -controls: Adds audio controls, like play, pause, and volume. -autoplay: Automatically starts playing the audio when the page loads. -loop: Repeats the audio once it ends. -muted: Mutes the audio by default. -preload: Specifies if and how the audio should be loaded when the page loads ('auto', 'metadata', 'none').
